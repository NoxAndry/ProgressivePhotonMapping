Программа реализует алгоритм прогрессивной фотонной трассировки для визуализации трехмерных сцен.
Сцена выставлена стандартной (cornell box), если надо - могу передать результаты рендеринга более сложных сцен.

Файлы l3ds и LoadBMP - внешние библиотеки для  загурзки 3d мэшей и BMP изображений соответственно.
