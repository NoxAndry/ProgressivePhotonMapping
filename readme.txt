Программа реализует алгоритм прогрессивной фотонной трассировки для фотореалистичной визуализации трехмерных сцен.
Написана совместно с Егором Ореховым.
Сцена выставлена стандартной (cornell box).

Файлы l3ds и LoadBMP - внешние библиотеки для загурзки 3d мэшей и BMP изображений соответственно.
